O que é a Licença Apache 2.0?

Visão Geral

A Licença Apache 2.0 é uma licença de software livre e de código aberto escrita pela Apache Software Foundation (ASF). É uma licença permissiva, o que significa que permite o uso, modificação, distribuição e uso comercial do software com poucas restrições.

Principais Características

  • Permite uso comercial: Você pode usar o software em projetos comerciais
  • Permite modificação: Você pode modificar o código-fonte conforme necessário
  • Permite distribuição: Você pode distribuir o software sob a mesma licença
  • Protege os contribuidores: Inclui uma cláusula de limitação de responsabilidade
  • Compatível com GPL: É compatível com a Licença Pública Geral GNU (GPL) versão 3.0

O que você pode fazer

Direitos Concedidos

Sob a Licença Apache 2.0, você tem o direito de:

  • Usar: Utilizar o software para qualquer propósito, incluindo comercial
  • Modificar: Alterar o código-fonte para atender às suas necessidades
  • Distribuir: Compartilhar cópias do software original ou modificado
  • Sublicenciar: Distribuir o software sob termos diferentes
  • Usar em software proprietário: Incorporar o código em projetos fechados

Obrigações

Como recipiente da licença, você deve:

  • Incluir uma cópia da licença em todas as distribuições
  • Manter o aviso de direitos autorais original
  • Incluir um arquivo NOTICE se fornecido com o software original
  • Documentar claramente as alterações feitas nos arquivos modificados

Perguntas Frequentes

Posso usar o DealeGear em projetos comerciais?

Sim! A Licença Apache 2.0 permite explicitamente o uso comercial do software. Você pode incorporar o código do DealeGear em seus produtos comerciais sem restrições.

Precoso abrir o código do meu projeto?

Não. A Licença Apache 2.0 não exige que você abra o código do seu projeto. Você pode manter seu código proprietário enquanto cumpre as obrigações básicas da licença.

Posso modificar a licença?

Não. Você não pode modificar os termos da Licença Apache 2.0. No entanto, você pode distribuir o software sob termos adicionais ou diferentes, desde que inclua os termos originais da licença.

Como devo atribuir o crédito ao DealeGear?

Você deve manter o aviso de direitos autorais original e incluir uma cópia da licença em todas as distribuições. Não é necessário atribuição adicional além disso, mas é apreciado.
