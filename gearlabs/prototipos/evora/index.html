Modo Virtual

Uma cidade construída com elementos virtuais, usando bibliotecas como ThreeJS para simular as interações humanas, com casas, escritórios, empresas, comércio, indústria, escolas, hospitais, dia e noite, mudanças de clima, evolução das pessoas (idade, renda, saúde, etc). É uma réplica do mundo real para estudo sobre as relações sociais. Serve tanto para uso em escolas e universidades como também para negócios.

Maquetes Funcionais com Microrrobótica
